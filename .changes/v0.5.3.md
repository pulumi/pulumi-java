## v0.5.3 - 2022-08-11

### Improvements

- [codegen] Enabled `pulumi convert` to target Java. Now a maven configuration (`pom.xml` with dependencies) is created and the generated Java files are moved to a proper package. [#702](https://github.com/pulumi/pulumi-java/pull/702)

### Bug Fixes

- [runtime] Reduced verbosity of Maven builds to only surface WARN and ERROR logs to the Pulumi user when Pulumi invokes Maven under the hood [#776](https://github.com/pulumi/pulumi-java/pull/776)
- [sdk] Fixed a serializer regression that affected the releases of `com.pulumi:kubernetes` after `3.19.1`. Upgrade `com.pulumi:pulumi` dependency to fix the issue that manifested as: [#782](https://github.com/pulumi/pulumi-java/pull/782)
