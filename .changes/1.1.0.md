## 1.1.0 - 2025-01-30

### Improvements

- [sdk] Avoid calling invokes with dependencies on unknown resources
- [sdk] Implement `InvokeOptionsBuilder` and `InvokeOutputOptionsBuilder`
- [codegen] Emit invoke options and invoke output options in generated programs
- [sdk] Include explicit dependencies of invokes in their resulting output
- [sdk] Bump core SDK versions in generated code
- [sdk] Emit plugin download URL in default resource options of the generated SDKs
- [sdk] Wait for resources in the input property dependencies

### Bug Fixes

- [codegen] Fix emitted functions for `assetArchive` and `remoteAsset` in generated programs
- [codegen] Fix generation of `double` literals in generated programs
- [sdk] Fix generation of untyped maps and array literals
- [sdk] Panic when asked to generate SDKs involving multi-argument invokes
